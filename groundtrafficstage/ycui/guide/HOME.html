# 🌈 开始

  • 此项目ycui-docs (opens new window)目前只包含用车组件库文档部分，组件通过外部引入，后期也可以整合到同一个项目中，方便管理

  • 暂时分项目开发B端和C端组件，添加组件时，先在对应端项目中添加，npm发布后，更新此文档，组件项目地址点击本页面右上角

  • 以下为此文档项目的开发说明：

# 说明

  • 项目由vuepress的1.x版本搭建，目前2.x处于beta版本，建议暂时不升级，具体可以查看vuepress官网 (opens new window)

  • 基于vuepress内置的布局和组件，可以很快的编写组件文档，md的形式非常便捷

  • 扩展了布局组件，方便C端组件的预览

# 写法

<source-block>
  <template slot="comp">
    <press-row>

        <yc-radio-check :data="[{label: '1', value: '1'}]"></yc-radio-check>
    </press-row>
  </template>

  </template>

# 引入外部依赖

如果要引入外部依赖，比如antd、elementUI等，在自定义主题theme中的enhanceApp.js文件，提供了vuepress当前正在使用的 Vue 构造函数和实例属性，可以注入和挂载相关属性

# 打包配置

引入一些必要的loader来进行编译，安装即可，注意所兼容的版本，vuepress已经内置了相关webpack配置，如要自定义可以参考下面链接：
